مبحث: رفع مشکل فارسی نویسی در قسمت ترمینال ویندوز

۰مقدمه و تشخیص مشکل

سلام! این راهنمای تعاملی به شما کمک می‌کند تا مشکل برعکس و جداجدا نوشته شدن حروف فارسی در ترمینال (مانند VS Code) را به صورت ریشه‌ای حل کنید.

دلیل اصلی مشکل: ترمینال پیش‌فرض قدیمی ویندوز (`cmd.exe`) پشتیبانی مناسبی از زبان‌های راست‌به‌چپ مثل فارسی ندارد. راه‌حل، نصب و تنظیم یک ترمینال مدرن است.

۱نصب ویندوز ترمینال (مهم‌ترین مرحله)

اولین و بهترین قدم، نصب **Windows Terminal** است. این ابزار مدرن که توسط خود مایکروسافت ساخته شده، به طور کامل از زبان فارسی پشتیبانی می‌کند.

نصب از فروشگاه مایکروسافت (Microsoft Store)

اگر به فروشگاه دسترسی ندارید، می‌توانید از صفحه گیت‌هاب آخرین نسخه را دانلود کنید (فایل با پسوند `.msixbundle`).

۲تنظیم به عنوان ترمینال پیش‌فرض ویندوز

حالا باید به ویندوز بگوییم که برای اجرای تمام دستورات، از این ترمینال جدید و قدرتمند استفاده کند.

  1. برنامه **Windows Terminal** را که نصب کردید، یک بار اجرا کنید.
  2. در بالای پنجره، روی فلش رو به پایین (▾) کلیک کرده و وارد **Settings** (تنظیمات) شوید.
  3. در تب **Startup** (راه‌اندازی)، گزینه **Default terminal application** را پیدا کنید.
  4. مقدار آن را از "Windows Console Host" به **"Windows Terminal"** تغییر دهید و تنظیمات را ذخیره (Save) کنید.

۳تنظیم به عنوان ترمینال پیش‌فرض در VS Code

در نهایت، باید به VS Code هم بگوییم که از این به بعد از پروفایل جدید برای ترمینال داخلی خود استفاده کند.

  1. وارد محیط VS Code شوید.
  2. کلیدهای ترکیبی `Ctrl` + `Shift` + `P` را فشار دهید تا نوار دستورات باز شود.
  3. عبارت زیر را تایپ و انتخاب کنید:
    Terminal: Select Default Profile
  4. از لیستی که باز می‌شود، گزینه‌ای که **PowerShell** را نشان می‌دهد (معمولاً با آیکون آبی) انتخاب کنید.
  5. تمام ترمینال‌های باز در VS Code را با کلیک روی آیکون سطل زباله ببندید.

۴آزمایش نهایی

کار تمام است! حالا وقت آزمایش است.

یک ترمینال جدید در VS Code باز کنید (با کلیدهای `` ` `` + `Ctrl`) و کد یا دستور فارسی خود را اجرا کنید. آیا خروجی به درستی نمایش داده می‌شود؟

۵مرحله نهایی: ترفندهای پیشرفته و راه‌حل‌های انجمن‌ها

اگر هیچ‌کدام از مراحل قبل جواب نداد، یعنی مشکل شما کمیاب و خاص است. در ادامه چند ترفند پیشرفته که از گفتگوهای برنامه‌نویسان در گیت‌هاب و فروم‌ها جمع‌آوری شده را به ترتیب بررسی می‌کنیم.

ترفند ۱: تغییر موتور رندرینگ ویندوز ترمینال

ویندوز ترمینال یک موتور رندرینگ جدید به نام `AtlasEngine` دارد که گاهی با برخی کارت‌های گرافیک یا فونت‌ها تداخل پیدا می‌کند. می‌توانیم آن را غیرفعال کنیم تا به حالت امن‌تر برگردد.

  1. **ویندوز ترمینال** را باز کرده و وارد **Settings** شوید.
  2. در منوی سمت چپ، روی پروفایل پیش‌فرض خود (معمولاً PowerShell) کلیک کنید.
  3. وارد تب **Advanced** شوید.
  4. در بخش **Rendering**، گزینه `Use the new text renderer ("AtlasEngine")` را روی **Off** یا **خاموش** قرار دهید و تنظیمات را ذخیره کنید.
  5. VS Code را کامل ببندید و دوباره باز کنید و تست کنید.

ترفند ۲: اجرای خودکار دستور `chcp 65001`

این دستور به ترمینال می‌گوید که از کدپیج `UTF-8` (استاندارد جهانی یونیکد) استفاده کند. می‌توانیم کاری کنیم که این دستور با باز شدن هر ترمینال در VS Code به صورت خودکار اجرا شود.

  1. در VS Code کلیدهای `Ctrl` + `,` را زده و آیکون فایل `settings.json` را باز کنید.
  2. کد زیر را پیدا کرده و به آن اضافه کنید. اگر وجود نداشت، کل قطعه کد را کپی کنید:
    "terminal.integrated.profiles.windows": { "PowerShell": { "source": "PowerShell", "icon": "terminal-powershell", "args": ["-NoExit", "-Command", "chcp 65001"] } }, "terminal.integrated.defaultProfile.windows": "PowerShell",
  3. فایل را ذخیره کنید، VS Code را ری‌استارت کنید و تست کنید.

ترفند ۳: نصب فونت مخصوص کدنویسی فارسی

گاهی مشکل از خود فونت است. فونت `Vazirmatn Code` یک نسخه ویژه با قابلیت `ligatures` است که برای محیط کدنویسی بهینه شده.

  1. از این لینک مستقیم از گیت‌هاب آخرین نسخه فونت وزیرمتن را دانلود کنید (فایل `Vazirmatn-Code-font-....zip`).
  2. فایل‌ها را از حالت فشرده خارج کرده، همه فونت‌ها را انتخاب، راست‌کلیک و **Install** کنید.
  3. در `settings.json` در VS Code، این دو خط را اضافه کنید:
    "terminal.integrated.fontFamily": "Vazirmatn Code", "editor.fontFamily": "Vazirmatn Code",

آخرین راه‌حل: استفاده از یک ترمینال کاملاً متفاوت

اگر هیچ‌کدام از روش‌های بالا جواب نداد، شاید بهتر باشد از یک برنامه ترمینال دیگر که موتور رندرینگ کاملاً متفاوتی دارد استفاده کنید. **Tabby** یک گزینه مدرن، زیبا و بسیار قدرتمند است که پشتیبانی خوبی از یونیکد دارد.

  1. از سایت رسمی Tabby آن را دانلود و نصب کنید.
  2. بعد از نصب، می‌توانید آن را جایگزین ترمینال داخلی VS Code کنید یا به صورت یک پنجره جدا از آن استفاده نمایید.
